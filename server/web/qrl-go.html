OpenQwaq Auto-launching Help

Most users have browser settings which allow the auto-launch process to happen automatically. However, there are some situations in which that may not happen. Please follow these directions if you do not seem to be automatically taken to the right place by your OpenQwaq client.

The auto-launch process works just like downloading a .doc file and double-clicking it to launch your word processor. OpenQwaq uses a type of file called a ".qwq" file for this.

  1. If you do not have a OpenQwaq account or do not have the OpenQwaq application installed, click here: First time user information.
  2. If your browser did not not automatically starting downloading a new file named "{!qwqFilename}" to your browser's download directory, click here: .qwq file download.
  3. If your browser does not open .qwq files automatically after download, you can open the downloaded file yourself by double-clicking it. Choose OpenQwaq as the application if prompted.
  4. If the OpenQwaq application is already running, but is hidden, minimized, or otherwise not visible, you may have to look among your open applications to see the OpenQwaq window.
  5. If OpenQwaq is already running and logged into a different organization, it will first ask you before leaving that forum or lobby.
  6. You may have to enter your username and password at the login screen.

Please contact {!contact} if you have any questions.

Thanks for using OpenQwaq.
